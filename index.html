basecubedev APT Repository

This is the official Debian/Ubuntu package repository for selected basecubedev open source projects.

This is a third-party APT repository. Only add it if you trust basecubedev as the package publisher.

Install repository key

sudo install -d -m 0755 /etc/apt/keyrings

curl -fsSL https://repo.basecubedev.de/basecubedev-archive-keyring.gpg \
  | sudo tee /etc/apt/keyrings/basecubedev.gpg > /dev/null

Add APT repository

echo "deb [signed-by=/etc/apt/keyrings/basecubedev.gpg] https://repo.basecubedev.de/debian stable main" \
  | sudo tee /etc/apt/sources.list.d/basecubedev.list

sudo apt update

Install packages

paping-go

sudo apt install paping-go

Project page: github.com/basecubedev/paping-go

Archive signing key

The repository metadata is signed with the basecubedev APT archive key. 

Fingerprint:
8F26 0E4D 9FD4 BDA3 9E28  547E 541A 3768 67EB B9BA

If the archive key is manually rotated, this fingerprint must be updated to match the public key served by the repository.

Remove repository

sudo rm -f /etc/apt/sources.list.d/basecubedev.list
sudo rm -f /etc/apt/keyrings/basecubedev.gpg
sudo apt update
